# KB Source Attestation

This document records the sources and contributors that informed the articles in this knowledge base. Articles were produced by converting or synthesizing source material into structured markdown using Claude. Most articles draw on more than one source type.

---

## Contributors & Sources

### Official REDCap Documentation

A significant portion of the KB was derived from REDCap's official online documentation, accessed as HTML pages from REDCap's built-in help system and public documentation site. Where a specific REDCap version is identifiable, it is noted in the individual article's metadata header (e.g., the AI Tools series references v15.5.3; FAQ-sourced content references v16.1.3).

### Bas de Veer — Original Training Materials

[Bas de Veer](https://www.linkedin.com/in/basdeveer), REDCap Manager, authored a set of training documents (`.docx` and `.pdf`) that served as primary source material for a number of articles. These are original instructional materials written from hands-on REDCap experience. Most of these documents include a REDCap version number, which reflects the version of REDCap the training material was written against. The version applicable to each article can be found in the individual article's metadata header. Source files are stored in `original docx files/`.

### St. Elisabeth Gruppe GmbH / Klinikum der Ruhr-Universität Bochum — MLM End User's Guide

The article **RC-MLM-01 — Multi-Language Management** was derived from the *Multi-Language Management End Users' Guide* (v15.5), kindly contributed by REDCap administrative staff at [St. Elisabeth Gruppe GmbH – Klinikum der Ruhr-Universität Bochum](https://www.ruhr-uni-bochum.de). This material is a community contribution to the REDCap Consortium and was used as the primary source for the MLM article.

### REDCap Control Center HTML Pages

Articles RC-EM-01 and RC-EM-02 were derived from the Control Center External Module Manager page (`ExternalModules/manager/control_center.php`), captured as an HTML file from a REDCap v16.1.3 installation. The source was reviewed and sanitized to remove all institution-specific references, URLs, usernames, and server names before knowledge extraction. Module descriptions are reproduced from the module metadata as displayed in the Manager UI, authored by the respective module developers. The institution-specific IRB Validator module was excluded from the catalog.

### Claude — Synthesized Articles

A subset of articles was generated by Claude by reasoning across existing KB articles rather than from an external source document. These articles cover topics where the relevant information was already distributed across the KB (for example, the Branching Logic in Longitudinal Projects article synthesizes content from the Branching Logic and Longitudinal series).

---

## Notes

- Many articles reflect a combination of the above — for instance, a training outline by Bas de Veer used as the primary structure, supplemented with detail from official REDCap documentation.
- Article version numbers in individual KB articles (`Version: 1.0`, `1.1`, etc.) reflect the KB article revision, not the REDCap software version.
- This attestation should be updated when new source types or contributors are introduced.
