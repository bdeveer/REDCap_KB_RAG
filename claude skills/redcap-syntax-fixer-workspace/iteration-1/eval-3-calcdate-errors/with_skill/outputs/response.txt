Expression type
@CALCDATE action tag on a Text Box field with date validation. The result field should have date_ymd (or date) validation applied.

Syntax errors

Cross-event reference syntax (position 1):
@CALCDATE([baseline_arm_1].[visit_date], 90, 'm')
                             ↑
The dot separator (.) is invalid for cross-event references. REDCap requires two adjacent bracket pairs with no separator between them. Should be [baseline_arm_1][visit_date].

Unit parameter (position 3):
@CALCDATE([baseline_arm_1][visit_date], 90, 'm')
                                              ↑
The unit 'm' (lowercase) means minutes, not months. To calculate 90 days after the baseline visit date, use 'd' (days). If you intended 90 months, use 'M' (capital M). Unit strings are case-sensitive.

Corrected expression

@CALCDATE([baseline_arm_1][visit_date], 90, 'd')
